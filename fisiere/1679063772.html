SUBIECT (90p)

n fiecare din cele cinci eprubete numerotate, aflate n stativul de pe masa de lucru, se gsete soluia uneia dintre urmtoarele substane:

Fe(NO 3 ) 2 , Fe(NO 3 ) 3 , Mn(NO 3 ) 2 , Al(NO 3 ) 3 , Zn(NO 3 ) 2 .

Identific ai substana aflat n fiecare eprubet pe baza reaciilor cu reactivii : NaOH, NH 3 , KI i K 2 CrO 4 , aflai pe masa de lucru n sticlue picurtoare.

n urma testelor efectuate, trecei rezultatele obinute ntr- un tabel identic cu tabelul 1, scriind n fiecare dreptunghi corespunztor interseciei dintre coloanele i liniile tabelului, dup caz:

  • formula compusului chimic rezultat, marcnd cu formarea unui precipitat i precizai culoarea lui; dac nu se formeaza precipitat, dar se formeaz un complex sau o alt specie creia i se datoreaz schimbarea culorii soluiei, precizai schimbarea culorii

  • marcai cu X dac n urma reaciei efectuate nu se observ nicio schimbare

  • Dac precipitatul este solubil n exces de reactiv, scriei formula compusului rezultat, iar dac precipitatul este insolubil, scriei aceast constatare n tabel.

  • Scriei ecuaiile reaciilor utilizate pentru identificarea substantelor din cele 5 eprubete in tabelul 2 din foaia de concurs.n cazul n care nu are loc reacie chimic, marcai un X n casua corespunztoare din tabel.
